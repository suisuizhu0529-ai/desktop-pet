# ⚡ 快速设置指南 - 5分钟快速上手

## 📋 前置要求

### Windows 用户
- Windows 10 或更新版本
- 已安装 Node.js (v14+)
  - 下载：https://nodejs.org/
  - 选择 LTS 版本
  - 安装时勾选 "Add to PATH"

### macOS 用户
- macOS 10.12 或更新版本
- 已安装 Node.js (v14+)
  - 方式1：https://nodejs.org/ 下载安装
  - 方式2：使用 Homebrew
    ```bash
    brew install node
    ```

### Linux 用户
- Ubuntu 18.04+ 或同等版本
- Node.js v14+
  ```bash
  sudo apt update
  sudo apt install nodejs npm
  ```

## 🚀 安装步骤

### 第1步：准备文件

1. 创建一个新文件夹，例如：`my-pet-app`
2. 将以下文件放入文件夹：
   - `main.js`
   - `preload.js`
   - `index.html`
   - `package.json`
   - `.gitignore`
   - `start.sh` (macOS/Linux)
   - `start.bat` (Windows)
   - `README.md`
   - `STRUCTURE.md`

3. 在同一文件夹创建 `assets` 子文件夹（用于存放图标）

**最终结构**：
```
my-pet-app/
├── main.js
├── preload.js
├── index.html
├── package.json
├── start.sh          (macOS/Linux)
├── start.bat         (Windows)
├── README.md
└── assets/
    └── (图标文件放这里)
```

### 第2步：打开终端/命令行

**Windows**：
1. 在文件夹空白处右键
2. 选择 "在此处打开命令窗口" 或 "在此处打开 PowerShell 窗口"

**macOS/Linux**：
1. 打开 Terminal（终端）
2. 输入命令进入文件夹：
   ```bash
   cd /path/to/my-pet-app
   ```

### 第3步：安装依赖

**所有系统**：

```bash
npm install
```

等待安装完成（可能需要几分钟）。

✅ 出现 "added XXX packages" 表示成功！

### 第4步：启动应用

#### 方式A：使用快速启动脚本（推荐）

**Windows**：
```bash
start.bat
```
或直接双击 `start.bat` 文件

**macOS/Linux**：
```bash
chmod +x start.sh
./start.sh
```

然后在菜单中选择 `1` 回车

#### 方式B：直接命令启动

```bash
npm start
```

### 第5步：测试应用

等待窗口出现（可能需要 20-30 秒）：

1. ✅ 应该看到一个小窗口，里面有宠物
2. ✅ 宠物上方有三个按钮
3. ✅ 右上角有一个⚙️按钮

**测试互动**：
- 点击 🤚 试试摸宠物
- 点击 🍖 试试喂宠物
- 点击 ⚙️ 打开控制面板

## 🎮 首次使用

### 打开控制面板

点击右上角的 ⚙️ 按钮

### 自定义宠物

1. 进入 🐾 宠物标签页
2. 修改宠物名字（在底部设置页）
3. 选择宠物形象（狗、兔子、狐狸、白虎）
4. 选择衣服（帽子、卫衣等）

### 配置工作收入（可选）

1. 点击 💰 工作标签页
2. 点击 "⚙ 修改配置"
3. 填入你的薪资信息：
   - 月薪：10000（元）
   - 月工作天数：22
   - 上班时间：09:30
   - 下班时间：18:30
4. 点击 "💾 保存"

### 添加待办清单

1. 点击 📋 待办标签页
2. 输入任务名称
3. 可选：设置截止时间
4. 点击 "＋ 添加"

### 创建计时器

1. 点击 ⏱️ 计时标签页
2. 输入计时器名称（如"喝水提醒"）
3. 选择时长
4. 点击 "＋ 添加计时器"
5. 点击 "▶ 开始" 开始倒计时

## 📦 打包成应用

当开发完成后，可以打包成可独立运行的应用。

### Windows 版本

```bash
npm run electron-build -- --win
```

生成的文件在 `dist/` 文件夹：
- `桌面宠物 Setup.exe` - 安装程序（推荐给他人使用）
- `桌面宠物.exe` - 便携版（无需安装）

### macOS 版本

```bash
npm run electron-build -- --mac
```

生成的文件在 `dist/` 文件夹：
- `桌面宠物-3.0.0.dmg` - 安装盘
- `桌面宠物-3.0.0.zip` - 压缩包

### 第一次构建

第一次构建可能需要几分钟，这是正常的。

✅ 看到 "Build complete" 表示成功！

## ❓ 常见问题

### Q1: "npm: 找不到命令"
**解决方案**：
- 重新安装 Node.js
- 重启电脑
- 检查环境变量

### Q2: 应用启动很慢
**解决方案**：
- 这是正常的，第一次启动需要初始化
- 之后会快很多

### Q3: 应用闪退了
**解决方案**：
```bash
# 清除缓存
npm cache clean --force
rm -rf node_modules
npm install
npm start
```

### Q4: 怎样修改宠物窗口大小
**解决方案**：
1. 用文本编辑器打开 `main.js`
2. 找到这一行：
   ```javascript
   width: 160,
   height: 200,
   ```
3. 修改数字后保存
4. 重启应用

### Q5: 如何卸载应用
**Windows**：
- 控制面板 → 程序和功能 → 找到"桌面宠物" → 卸载

**macOS**：
- 打开 Finder → Applications → 找到"桌面宠物" → 拖到废纸篓

## 📞 获取帮助

如遇到问题，请按以下顺序排查：

1. **检查 Node.js**
   ```bash
   node -v
   npm -v

   rm -rf node_modules package-lock.json
   npm install
   ```

   - 在终端中仔细查看错误信息
   - 记下错误代码
   - 搜索相关问题

4. **重启电脑**
   - 有时能解决奇怪的问题
